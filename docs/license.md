# UPLim Licensing

This document defines the project licensing model for UPLim.

## Canonical License

UPLim is released under the MIT License.

The canonical license text for the repository is:

- [LICENSE](/Users/tarashuliichuk/MyProjects/UPLim/LICENSE)

## What Is Covered

Unless a file or subdirectory explicitly states otherwise, the MIT License applies to:

- the UPLim language implementation
- the compiler toolchain
- the runtime and compatibility layers
- the standard library source code
- examples and test fixtures in this repository
- the language specification and project documentation in this repository

## Practical Meaning

The MIT License allows users to:

- use UPLim commercially
- modify the compiler and runtime
- redistribute source or binaries
- create private forks
- create proprietary software in UPLim

The main requirement is to preserve the copyright notice and the license text.

## Language And User Code

The MIT License applies to the UPLim project itself, not automatically to every program written in UPLim.

Programs written in UPLim may use any license chosen by their authors unless they copy material from this repository that remains under MIT.

## Future Policy

If a future component needs a different license, it must declare that explicitly in its own directory and the repository documentation must be updated to point to that exception.

Until such an exception exists, MIT is the single source of truth for UPLim licensing.
