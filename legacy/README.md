# Legacy Areas

This directory contains historical prototypes that are no longer part of the active UPLim package graph.

Active development happens in:

- `packages/core`
- `packages/cli`
- `packages/lsp`
- `apps/website`
- `apps/vscode-extension`

`legacy/tooling` is kept only for reference while the new monorepo stabilizes.
`legacy/tooling/core-prototype` contains the archived analysis/LSP/AI prototype files removed from the active `packages/*` graph during the internal package split.
`legacy/web-prototype` contains the former root-level Next.js dashboard/playground surface.
`legacy/examples/uplim-website-prototype` contains an older UPLim-language website experiment.
