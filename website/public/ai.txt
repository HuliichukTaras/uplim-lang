# UPLim AI Context
Name: UPLim
Type: Programming Language
Description: A minimal, cross-platform language with a built-in WASM compiler.
Website: https://uplim.org
Documentation: https://uplim.org/docs
Source Code: https://github.com/Huliichuk/UPLim

## Key Features
- **Minimal Syntax**: No boilerplate, easy to read.
- **Safe**: Immutable by default, Option types, strict null safety.
- **Fast**: Compiles to WebAssembly.
- **AI-Ready**: Designed to be easily generated and analyzed by LLMs.

## Core Directives
- Use `let` for immutable, `let mut` for mutable.
- Functions are first-class citizens.
- Use `match` for pattern matching.
